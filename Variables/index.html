Declaring Variables

Variables are declared using the keywords var, let, const //var & let declare varaibles that can change over time
var beef;
let lastName;

const TAX; //this declares a constant variable

Uninitialzed variables have a default value of undefined console.log(car); //undefined

Assigning Variables

Variables are asssigned values using the assignment operator(equals sign "=") car = "Volvo";
beef = "Bokoto";
myName = "Babajide";
const VAT = 7.5;
const COUNTRY_CODE = "NGR";

Case Sensitivity

Javascript variables are case sensitive.So, Mother, mother, and moTheR are not the same variables

var Mother = 'Janet';
console.log(Mother); //-> Janet Mother = 'Mary';
console.log(Mother); //-> Mary MoTher = "Suzan"; // -> Error

Diffrence between var and let

1. Variables declared with var can be accidentally redclared.
2. Unlike var variables declared with "let" can only be declared once.
3. Any attempt to redclare a varable declared with "let" will result in Syntax Error var player = 'James';
var player = 'Peter';
console.log(player); //-> Peter No Errors let player = 'James';
let player = 'Peter'; //-> Error!!

Declaring Read Only Varaibles

We can decalare Read Only variables with the keyword "const". const has all the benefit of the "let" keyword with the added bonus that varables const are read only and can not be reassigned once declared.

const FAV_PET = "Dogs"; FAV_PET = "Cats"; //-> Error!!
