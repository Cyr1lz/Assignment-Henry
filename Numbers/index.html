Using Numbers with JavaScript

Number is a data type in JavaScript which represents numeric data

Add two Numbers with JavaScript
const myVar = 4 + 8;
console.log(myVar); //-> 12
Subtract two Numbers with JavaScript
myVar = 8 - 4;
console.log(myVar); //-> 4
Multiply two Numbers with JavaScript
myVar = 8 * 4;
console.log(myVar); //-> 32
Divide One Number by Another with JavaScript
myVar = 8 / 4;
console.log(myVar); //-> 2
Increment a Number with JavaScript
let i = 0;
i++; //-> 1 - Increment Operator

is the equivalent of

i = i + 1; //-> 2
Decrement a Number with JavaScript
i = 2;
i--; //-> 1 - Decrement Operator

is the equivalent of

i = i - 1; //-> 0
Create Decimal Numbers with JavaScript
let decimalNum = 2.75; // Also referred to as a floating point number
Multiply two Decimal Numbers with JavaScript
console.log(45 * 2); // 90
Divide one Decimal by another with JavaScript
console.log(45 / 5);
Finding the remainder with JavaScript
console.log(5 % 2); // 1
