Javascript Strings

Using Strings with JavaScript

Strings in Javascript are a sequence of Unicode characters enclosed within quotation marks, which represent textual data.

Declaring a String

A string may be decalred with either single or double quotes.

let myStr = "this is a string declared with double quotes"; myStr = 'this is a string declared with single quotes';
Escaping Literal Quotes in Strings
myStr = "there is \"an escaped quote literal\" inside this string" console.log(myStr); //-> there is "an escaped quote literal" inside this string
Concatenating Strings with Plus operator
/* Watch out for spaces. Concatenation does not add them */ let concatenatedStr = "I am a string " + "I'm concatenated " + "with other strings." console.log(concatenatedStr) //-> "I am a string concatenated with other strings"
Concatenating Strings with Plus Equals operator
concatenatedStr += " using the Plus Equals operator" console.log(concatenatedStr) //-> "I am a string concatenated with other strings using the Plus Equals operator"
Finding the Length of a string
/* Note Everything in JavaScript is an object, and every object has properties and methods. Therefore to find the length of a string for example we use the 'length' property of the string*/ let lengthOfStr = output.length; console.log(lengthOfStr); //-> 86 includes the spaces in between.
